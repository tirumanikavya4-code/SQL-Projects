# Hospital Data Analysis using SQL

## Project Overview
This project demonstrates SQL skills by analysing hospital data to answer real business questions.

## Skills Used
- SQL
- SELECT
- WHERE
- GROUP BY
- ORDER BY
- JOINS
- Aggregate Functions
- CASE Statements

## Business Questions
1. Which department has the highest number of patients?
2. Which doctor treats the most patients?
3. What is the average patient age?
4. Which diagnoses are most common?
5. How many admissions occur each month?

## Tools
- MySQL
- SQL Server
- GitHub

## Outcome
This project demonstrates practical SQL querying and analytical skills relevant to healthcare reporting and business intelligence roles.
